## Reporting a Vulnerability

We take the security of this personal study project seriously. If you discover a vulnerability, please do not disclose it publicly.

### How to Report
Please report security vulnerabilities by creating a **Private Issue** on the repository or by contacting the maintainer via GitHub profile.

### What to Expect
- **Response Time**: You can expect an acknowledgment of your report within 48-72 hours.
- **Updates**: We will provide status updates at each stage of the investigation and remediation process.
- **Remediation**: Once a vulnerability is verified, a fix will be implemented and pushed to the `main` branch.
